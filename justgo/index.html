01 / THE IDEA

“A route can be technically possible
and still be practically impossible.”

JustGo uses route-picker baselines where available, asks Apple Maps for place search and walking legs, and labels each station field by its actual source. The result is guidance with reasons, caveats, and visible data boundaries—not a mysterious green score.

High confidence · 80–100Fewer concerns in the available data
Worth checking · 60–79One or more details need attention
Plan carefully · 0–59Complex, risky, or data-limited

02 / MAIN f35e624 · JULY 2026

More than
A to B.

The current app has grown far beyond the original six-city concept.

01

Track-shaped route previews

Bundled routes project station endpoints onto the selected OpenStreetMap-derived track path, then draw only the traveled segment.

02

Map-first Live Go

A full-screen interactive map shows the route and user position. Repeated off-route readings during a walking step can trigger replanning.

03
◎ONE TAP

Quick Tags & replayable tour

Home, Work, and custom station tags fill the active planner field in one tap; the four-page walkthrough can be replayed from Settings.

04
✣

05

Live only where stated

Hong Kong has live-arrival references for all 162 matched stations. Viewing live results contacts the government transport API; those results are not offline content.

06

Personal, on your device

Personal station media remains entirely on-device and is never used as verified routing, transfer, platform, or door data.
